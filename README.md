# Cointegration
This repository explores the concept of cointegration to uncover long-run relationships in time series data.

**Cointegration**: Uncovering Long-Run Relationships in Time Series  This project, developed by **Team "Big 4"** at the **Ramakrishna Mission Vivekananda Educational and Research Institute (RKMVERI) Belur**, during the **Semester 3 of the M.Sc. BDA programme**, explores the statistical methodologies used to analyze non-stationary time series data.  

## Key Topics Covered:
- *Time Series Fundamentals*: Analyzing stationary processes, random walks, and testing for unit roots using the Augmented Dickey-Fuller (ADF) test.
- *The Spurious Regression Trap*: Understanding how regressions between independent random walks can falsely show high correlations and shared trends rather than true economic relationships.
- *Cointegration*: Identifying long-run equilibrium relationships between non-stationary variables.
- *Modeling & Testing*: Utilizing Error Correction Models (ECM), Vector Error Correction Models (VECM), and the Johansen test (including Trace and Max-Eigenvalue statistics) to determine the number of cointegrating relationships.
